## Example Config File

We provided you with an example configuration file if needed.

```lua tile="config.lua"
--!strict
local Typings = require(game:GetService'ReplicatedStorage'.Spawner.SpawnerTypings);

--[[

ONLY EDIT THE CONFIG TABLE!

Need help? Visit https://docs.tyra.digital/docs/rbx-products/bus-spawner/configuration/

]]

local config: Typings.Config = {
	theme = {
		title = "My Spawner",
		colorScheme = "dark",
		logoImageId = 9428951785,
	},

	categories = {
		Category1 = {
			title = "Spawn Bus",
			categoryType = "open",
		},
	},

	groupSettings = {
		enabled = false,
		groupId = 13299030,
		minimumSpawnerRank = 1,
		minimumAdminRank = 255,
		allowVipOwnersServerAdmin = false
	},

	busModelsLocation = game.ServerStorage.BusModels,
	depsawnOnPlayerLeave = false,
	teleportPlayerToBus = false,
	busModelSpawnType = "basic"
}

return config;
```

| [theme](#theme-configuration-options)         | ✔️       | Includes all the data for your theme such as a title, custom logo image, and color themes! (Blue & Dark Theme are the best :sunglasses:) |    [View this for value choices](#theme-configuration-options) |              |

| [groupSettings](#group-configuration-options) | ✔️       |                                Includes all the data about your group and group settings for the spawner.                                |    [View this for value choices](#group-configuration-options) |              |
| busModelsLocation                             | ✔️       |                                       If you want to add more spawner categories (up-to 3 extra).                                        |                Object with `cateogryTwo` as `false` \| `false` |      `false` |
