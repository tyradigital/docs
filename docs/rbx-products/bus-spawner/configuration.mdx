## Example Config File

```lua tile="config.lua"
-- VIEW https://docs.tyra.digital/guides/roblox-products/bus-spawner/configs FOR MORE DETAILS

return {
	defaultSettings = false,

	title = 'My Groups Spawner',
	theme = 'light',
	despawnOnLeave = true,
	extraSpawnerCategories = {
		categoryTwo = 'Secondary Spawn',
	}
}
```

## Configuration Options

### Basic Configuration Options

| Name                   | Required |                                                                                                          Description                                                                                                          |                                   Value Choices | DefaultValue |
| :--------------------- | :------- | :---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------: | ----------------------------------------------: | -----------: |
| DefaultSettings        | ✔️       | If you want the spawner to pick the default settings, you can look in the main file, but please do not edit it to see the default settings. If it is set to **true**, then nothing set in your config file will be in effect. |                               `true` \| `false` |      `false` |
| Title                  | ✔️       |                                               If the name at the top of your spawner. You could name this your group name, or just name it 'My Neat Bus Spawner' if you'd like.                                               |                                        `string` |  `"Spawner"` |
| Theme                  | ❌       |                                                                       f you want to have it be light or dark theme (Psst, dark is cooler :sunglasses:).                                                                       |                           `"light"` \| `"dark"` |    `"light"` |
| DespawnOnLeave         | ✔️       |                   If you want the spawner to despawn someones bus after the exit the server/game. This is useful to prevent unnessecary lag and so you don't have to btools somebody's bus when they leave.                   |                               `true` \| `false` |       `true` |
| ExtraSpawnerCategories | ✔️       |                                                                                  If you want to add more spawner categories (up-to 3 extra).                                                                                  | Object with `cateogryTwo` as `false` \| `false` |      `false` |
