title: Clothing
---

Example Configuration:

```lua
{
	title = "My Category",
    // highlight-next-line
	categoryType = "clothing",
    // highlight-next-line
    assetId = 12345678,
}
```

To set a category to a clothing item - set the `categoryType` to `"clothing"`

Then, set your `assetId` to the asset Id of the clothing article
